Compose ton Logo est un moyen simple de s'essayer à la création graphique. À l'aide d'une borne, les visiteurs pourront créer leur logo en utilisant des contrôles simplifiés inspirés des logiciels tels que Figma ou Illustrator.

Il a été créé en groupe avec Roxane Omnes, qui s'est occupée des recherches UX et de la conception de l'habillage.

[Essayer en ligne »](https://compose-ton-logo.vincelinise.com/) • [Page descriptive du projet »](https://vincelinise.com/projects/compose-ton-logo)

## Stack technique
Compose ton Logo utilise [React.js](https://react.dev/), [Tailwind.css](https://tailwindcss.com/), [Electron Vite](https://electron-vite.org/) et [Framer Motion](https://motion.dev/). Il est conçu pour fonctionner dans un environnement de bureau avec 2 écrans distincts connectés, mais une version web de démo est également disponible.

## Commandes de développement et de build
### Prévisualisation de développement
```sh
npm run dev
```
### Compiler pour Windows (64 bits)
```sh
npm run package
```
### Empaqueter la version Web de démo
```sh
npm run build:web
```
